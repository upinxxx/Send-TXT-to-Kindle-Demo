syosetu2epub.online – Send TXT to Kindle Demo

Service Description

Users must first add noreply@syosetu2epub.online to their Kindle's Approved Personal Document E‑mail List.

Users upload a TXT file → The system converts the text to EPUB format → The EPUB file is sent as an attachment to the user's Kindle email address via AWS SES, sending only 1 email at a time.

Usage Example
